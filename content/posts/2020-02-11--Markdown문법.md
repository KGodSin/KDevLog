# 마크다운(MarkDown)이란?
파일 확장자 이름이 .md로 된 파일이다.
보통 마크다운이라는 문법을 통해서 쉽고 빠르게 문서 작성이 가능하다는 장점을 가지고 있다.

# 문법 배우기
## Header
```<h1>```부터 ```<h6>```까지 표현 할 수 있다.
``` markdown
	# h1
	## h2
	### h3
	#### h4
	##### h5
```
